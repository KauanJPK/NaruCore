# NaruCore
NaruCore é um plugin para Spigot com compatibilidade e versões universais, um projeto em desenvolvimento, conectado ao projeto Kotlinaru.

## Funcionalidades

### Conexão direta com o bot Kotlinaru
-Capaz de aumentar a comunicação *Discord <-> Minecraft*

### Moderação
- Banir e expulsar usuários.
- Silenciar e gerenciar permissões.
- Comandos de gerenciamento de servidor.

### Comandos Gerais
- Ping, ajuda, informações do servidor e do usuário.

### Extensibilidade
- Fácil de adicionar novos comandos e módulos.
- Estrutura modular organizada por serviços e listeners.
